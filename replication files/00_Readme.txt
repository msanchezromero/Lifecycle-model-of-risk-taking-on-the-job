This folder contains R scripts that replicate the empirical analyses from the raw data.

Data download:
The raw data must be obtained manually and placed in the "raw" subfolder.
Instructions for downloading the data are provided at the beginning of scripts 01_* to 03_*.

Preprocessing:
Scripts 01_* to 03_* handle data preparation and cleaning. The edited data is stored in the "data" subfolder.

Analysis:
Scripts 04_* and 05_* perform the main empirical analysis.

Output:
Figures generated by the analysis are saved in the "figures" subfolder.
